Team Collaboration

This project was developed as a collaborative effort between two developers as part of a backend development training project.

The development process followed Agile principles, with tasks divided into small actionable units called User Stories.

Each feature was implemented through pair collaboration and structured task management.

🧩 Project Management

To organize development and track progress, we used:

Jira

Jira was used to:

create and manage tickets
track user stories
divide tasks between team members
monitor development progress
identify blockers during development

The workflow included:

Backlog

All features were first written as User Stories.

Example:

US1 – User Registration
US2 – User Login
US3 – Display My Recipes
US4 – Create Recipe
US5 – Edit Recipe
US6 – Delete Recipe
US7 – Recipe Categories
US8 – Filter by Category
Task Assignment

Each feature was broken into technical tasks and assigned through Jira tickets.

Example tickets:

Create Database Schema
Implement User Authentication
Develop Recipe Model (CRUD)
Build Recipe Controller
Design Recipe Index View
Add Category Filtering
Implement Favorite Recipes
Development Workflow

The team followed a simple Agile cycle:

1️⃣ Create ticket in Jira

2️⃣ Assign task to a team member

3️⃣ Implement the feature

4️⃣ Push changes to GitHub

5️⃣ Review and test together
